# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer behavior, and product performance.

## Objectives
- Clean and preprocess retail sales data
- Perform descriptive statistics
- Analyze sales trends over time
- Understand customer and product behavior
- Provide business-focused recommendations

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified high-performing product categories
- Observed seasonal sales trends
- Analyzed customer purchasing patterns
- Found strong correlation between quantity and revenue

## Business Recommendations
- Focus on high-performing product categories
- Optimize inventory during peak sales periods
- Target high-value customer segments
- Improve performance of low-selling products

## Files in This Repository
- `Retail_Sales_EDA.ipynb` – Jupyter Notebook with full analysis
- `retail_sales_dataset.csv` – Dataset used for analysis
